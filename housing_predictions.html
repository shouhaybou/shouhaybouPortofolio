Housing Data Prediction Project

This project involves building a predictive model to forecast housing prices using a variety of data features. The project includes data preprocessing, feature engineering, model training, and evaluation. The model aims to provide accurate predictions to assist in making informed decisions in the real estate market.

Project Details

Technologies Used: Python, Scikit-learn, Pandas, NumPy
Data Source: Github
Key Features:

  • Data cleaning and preprocessing
  • Exploratory data analysis and feature engineering
  • Model training with various regression algorithms
  • Model evaluation and selection

HOUSING_PATH = os.path.join("datasets", "housing")
HOUSING_URL = DOWNLOAD_ROOT + "datasets/housing/housing.tgz"

# Method to download the data
def fetch_housing_data(housing_url=HOUSING_URL, housing_path=HOUSING_PATH):
    if not os.path.isdir(housing_path):
        os.makedirs(housing_path)
    tgz_path = os.path.join(housing_path, "housing.tgz")
    urllib.request.urlretrieve(housing_url, tgz_path)
    housing_tgz = tarfile.open(tgz_path)
    housing_tgz.extractall(path=housing_path)
    housing_tgz.close()
fetch_housing_data(HOUSING_URL, HOUSING_PATH)
