Brain Tumor Detection Project

This project involves the development of a machine learning model to detect brain tumors from MRI scans. The model leverages advanced deep learning techniques to accurately classify images as either containing a tumor or not. The project includes data preprocessing, model training, and evaluation phases, ensuring high accuracy and reliability.

Project Details

Technologies Used: Python, TensorFlow, Keras, OpenCV
Data Source: Kaggle
Key Features:

  • Data preprocessing and augmentation
  • Custom convolutional neural network (CNN) architecture
  • Model evaluation and validation
  • F1-score computed on the test dataset
